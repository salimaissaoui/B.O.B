# Schematics Folder

Place your `.schem` or `.schematic` files here.

## Usage
```
!build C:/repos/B.O.B/schematics/castle.schem
```

Or use `--at` to specify build location:
```
!build C:/repos/B.O.B/schematics/house.schem --at 100,64,100
```

## Supported Formats
- `.schem` (WorldEdit/Sponge format)
- `.schematic` (MCEdit format)
- `.nbt` (Structure Block format)

## Where to Find Schematics
- [Cubical.xyz](https://cubical.xyz/) - Browser-based 3D editor with schematic export
- [Planet Minecraft](https://www.planetminecraft.com/projects/tag/schematic/)
- [Minecraft Schematics](https://www.minecraft-schematics.com/)
